Introduction
=============
Wallet SDK C++ APIs, you can use these apis to implement an offline wallet.

Single address wallet, HD wallet, generate mnemonic, recover wallet by mnemonic, send transaction, cross chain transaction, set/get did infos, recover did by mnemonic. All these feature are included.

.. warning::
    Since software is written by humans, it can have bugs. Thus, if you have questions you are always welcome to issue your problem.
